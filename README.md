# Flight Tracker Wallpaper

Live flight radar as your desktop wallpaper.

## Quick Start (One Step)

**Mac:** Double-click `LAUNCH.command`  
**Windows:** Double-click `LAUNCH.bat`

That's it. A fullscreen flight tracker window opens automatically.

First run installs one dependency (pywebview) — takes ~10 seconds.

## Auto-Start on Boot

Run once to make it launch every time you log in:

```
python3 autostart.py
```

To remove from startup:
```
python3 autostart.py --remove
```

## Controls

- ESC or Cmd+Q / Alt+F4 to close
- Scroll to zoom the map
- Drag to pan
- Hover on planes for flight info

## Requirements

- Python 3 (pre-installed on Mac; get from python.org for Windows)
- Internet connection
